# 요구사항명세서 (SRS)

**프로젝트:** 경비처리 웹서비스
**버전:** 1.0 · **작성일:** 2026-07-14

---

## 1. 개요

### 1.1 배경
회사 경영지원실은 매월 전사 경비를 취합·분류하고 세무 신고 자료를 준비한다. 실무에서는
증빙(영수증·세금계산서·카드전표)을 모으고, 계정과목·부서로 분류하고, 부가세 공제 여부를 판정한 뒤,
세무대리인에게 엑셀과 증빙을 전달한다. 이 과정은 대부분 수작업이며 오류·누락이 잦다.

### 1.2 목적
- 직원이 경비를 **손쉽게 등록**(영수증 촬영 → OCR 자동 인식)하도록 한다.
- 경비를 **소속 조직·계정과목·적격증빙 기준으로 자동 분류**한다.
- 승인 워크플로우로 **통제·추적성**을 확보한다.
- 월 마감 시 **세무 신고 자료(엑셀 + 증빙 ZIP)를 원클릭 생성**한다.

### 1.3 용어
| 용어 | 정의 |
|---|---|
| 적격증빙 | 세법상 인정되는 지출증빙 — 세금계산서·계산서·신용카드매출전표·지출증빙용 현금영수증 |
| 비적격증빙 | 간이영수증 등. 3만원 초과 시 증빙불비가산세(2%) 대상 |
| 매입세액공제 | 부가가치세 신고 시 매입 부가세를 공제받는 것 |
| 귀속월 | 경비가 귀속되는 회계 월(YYYY-MM) |
| 마감 | 검토 완료된 경비를 확정·잠금하고 산출물을 생성하는 행위 |

## 2. 이해관계자 · 사용자 역할

| 역할 | 설명 | 주요 권한 |
|---|---|---|
| **일반 직원 (employee)** | 경비를 사용·신청하는 구성원 | 본인 경비 신청·수정·삭제·제출, 본인 내역 조회 |
| **팀장 (manager)** | 부서/팀 관리자 | 소속 팀 경비 승인/반려, 소속 팀 내역 조회 |
| **경영지원실 (admin)** | 경비/세무 담당 | 전사 조회, 검토·마감, 엑셀/증빙 산출, 조직·사용자 관리 |

## 3. 기능 요구사항 (FR)

### FR-1 인증 · 권한
- FR-1.1 이메일/비밀번호 로그인, JWT(Access+Refresh) 발급
- FR-1.2 역할 기반 접근 제어(직원/팀장/관리자), 열람 범위 제한(본인/소속 팀/전사)
- FR-1.3 Access Token 만료 시 Refresh Token으로 자동 재발급

### FR-2 조직 · 사용자 관리 (관리자)
- FR-2.1 회사 › 부서 › 팀 트리 조회
- FR-2.2 부서/팀 등록·삭제 (팀이 있는 부서·직원이 있는 팀은 삭제 차단)
- FR-2.3 사용자 등록, 역할·소속 팀·활성 여부 수정

### FR-3 경비 신청
- FR-3.1 신청서(제목·귀속월) + 다건의 경비 항목 등록
- FR-3.2 항목 필드: 사용일자·거래처·공급가액·부가세·합계·계정과목·증빙유형·결제수단·적요·증빙 이미지
- FR-3.3 **소속 부서/팀 자동 배부**(등록 시점 스냅샷, 이후 인사이동에도 귀속 유지)
- FR-3.4 작성중/반려 상태에서 본인 신청서 **수정·삭제**

### FR-4 영수증 OCR
- FR-4.1 영수증 이미지 업로드 → 거래처·사업자번호·날짜·합계금액 **자동 추출**
- FR-4.2 인식 실패/저신뢰 시 **수동 입력으로 자연스러운 폴백**
- FR-4.3 OCR 성공 여부와 무관하게 **원본 이미지 저장·첨부**

### FR-5 도메인 규칙 (자동 적용)
- FR-5.1 적격+과세 증빙은 합계에서 **공급가액/부가세 자동 분리**, 면세·비적격은 전액 공급가액
- FR-5.2 적격+과세 증빙 & 공제대상 계정일 때만 **부가세 공제** 처리(접대비·비영업용 차량 등은 불공제)
- FR-5.3 **3만원 초과 비적격 증빙 경고**(증빙불비가산세 리스크)
- FR-5.4 공급가액 + 부가세 = 합계 **금액 일치 검증**

### FR-6 승인 워크플로우
- FR-6.1 상태 전이: `작성중 → 제출 → 팀장승인 → 검토완료 → 마감`
- FR-6.2 반려(사유 필수) 시 작성중으로 복귀, 재제출 가능
- FR-6.3 팀장은 소속 팀 신청서만 승인/반려
- FR-6.4 모든 상태 변경(제출·승인·반려·검토·마감)을 **이력으로 기록**하고 타임라인 표시

### FR-7 월 마감 · 산출물
- FR-7.1 검토 완료된 경비를 귀속월 단위로 마감·잠금
- FR-7.2 **세무 신고용 엑셀** 자동 생성 — 경비내역 / 부서·팀별 집계 / 계정과목별 집계 / 부가세 정리
- FR-7.3 마감 증빙 이미지를 **ZIP으로 묶어 다운로드**(부서_거래처_일자 파일명)

## 4. 비기능 요구사항 (NFR)

| 구분 | 요구사항 |
|---|---|
| **보안** | 전 구간 인증 필수, 비밀번호 bcrypt 해시, JWT 서명, 자격증명은 환경변수로만 주입(코드/저장소 미포함) |
| **개인정보** | 최소 수집, 국내 리전 보관(운영), 카드번호 등 민감정보 미저장 |
| **증빙 보관** | 세법상 **5년 보관**을 고려한 스토리지 정책(운영 S3 + 수명주기/Object Lock) |
| **접근통제** | RBAC 3역할, 열람 범위 분리, 증빙 이미지 열람도 신청서 권한 확인 |
| **감사성** | 승인·반려·마감 이력 영속 기록 |
| **성능/확장** | OCR·스크래핑 등 외부 I/O는 비동기 워커로 분리 가능한 구조(운영 확장 시) |
| **이식성** | 개발=SQLite, 운영=PostgreSQL 동일 코드(제네릭 타입·Alembic) |
| **테스트성** | 도메인 규칙·API·워크플로우를 자동 테스트(pytest 55) |

## 5. 도메인 규칙 · 제약

- **적격증빙 4종**: 세금계산서 / 계산서(면세) / 신용카드매출전표 / 지출증빙용 현금영수증
- **3만원 규칙**: 건당 3만원 초과 지출은 적격증빙이 없으면 증빙불비가산세 대상
- **부가세 공제 제외**: 기업업무추진비(접대비), 비영업용 승용차 관련비, 면세 매입 등
- **세무 일정(참고)**: 원천세(매월 10일), 부가세(분기 25일), 법인세(3월) — 본 서비스는 **세무대리인 전달 자료 생성**까지를 담당

## 6. 범위

### 포함
- 경비 등록·분류·승인·마감, OCR, 증빙 보관, 엑셀/ZIP 산출, 조직·사용자 관리

### 제외 (의도적)
- **홈택스·카드사 자동수집(스크래핑 벤더 연동)** — 인증정보 위탁·보안 부담을 고려해 1차 범위에서 제외.
  대신 최종 산출물을 세무대리인 전달용 엑셀+증빙으로 제공하여 실무 흐름을 충족.
- 급여·4대보험(경비와 분리 영역), 자체 홈택스 전자신고

### 향후 확장 후보
- 대시보드 집계/차트, S3 스토리지 전환, OCR Receipt 특화 도메인, 홈택스/카드 자동수집, AWS 배포 IaC
